# Preface
I don't use `mpv` much, but when I do use it, I don't want an ugly, unusable UX to greet me, thus this section.

Aside from that, I also run `mpv` using the `vulkan-video` (soon landing in browsers, trust) backend for hwdec.

## How to?
Simple - copy and tweak in your favorite text editor whichever component you want to use:
- [modernz](https://github.com/Samillion/ModernZ) is the UI and font component
- [thumbfast](https://github.com/po5/thumbfast) is the thumbnail generation component
- `input.conf` is the list of my keybinds
- `mpv.conf` tweaks the behavior of mpv (if you want nothing from here, but the UI, you only need `line 1` from here)
